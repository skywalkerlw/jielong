# How to run

1. install `vapor` cli via https://vapor.codes/
2. run `vapor build`
3. run `vapor run`
4. run `http://127.0.0.1:8080/front/basic/live`, show `i am live`
5. create a databas in mysql named `oz_omega` and create a table named `o_gallery` with `name` and `description` field.
6. run command line `curl --location --request POST 'http://127.0.0.1:8080/front/gallery/apply' \
--form 'name="liang wang"'
7. that's all
